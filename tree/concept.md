##  concept

* 树的度
max of all 结点的度

* 结点的度
son node of current node

* 分支节点/非终端节点
include root node and 内部节点
度 != 0

* 叶结点/终端结点
度 == 0

* 结点的祖先
all node from root node to parent node of current node

* 节点的子孙
**all** son node of it

* 节点的层次
root node is 1 layer

* 树的深度/树的高度
max of all 结点的度

* 有序树
各子树 从左到右 有次序

* 森林
collection of tree which 互不相交
