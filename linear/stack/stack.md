##  stack

only can add to top and delete from top

###   array
```shell
			add()   get()
				\   ^
				 v /
		 _		_____
		/		|	|
	   /		|	| <- top
 size()		   /| x |
	  \	  len()	| x |
	   \	   \| x | <- bottom
		\-		-----
```


###   link
```shell
[头指针]-->	[首元节点] (top element)
				|
				v
			[第二节点]
				|
				v
			   ...
```
