##  邻接表
有向图/无向图 使用
适合边 远少于 顶点


###   结构
数组/链表 ： 存储**顶点**
多个链表  ： 一个数组的元素链接0~1个链表，链表是顶点的所有 **出弧/边**

```shell
v0[data][第一个边的指针] -> [顶点的索引][weight][下一个边的指针] ->
v1[data][第一个边的指针] -> [顶点的索引][weight][下一个边的指针] ->
v2[data][第一个边的指针] ->
...
vm[data][第一个边的指针] ->
vn[data][第一个边的指针] ->
```

##  逆邻接表
和邻接表差不多，但顶点的边链表是所有 **入弧**
