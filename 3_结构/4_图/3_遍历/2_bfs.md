##  bfs
广度优先
像 树的层序遍历 ： 使用队列



###   步骤
连通图 : 做一个 dfs
非连通图 : 做完 dfs 后, check `visited[]`, 如果还有顶点未访问，重复做 dfs ...

1. for 初始化 `visited[]` 
2. for 对每个未访问的顶点做 bfs()
3. bfs() : 标记 `visited[当前顶点]` 。当前顶点入队
4. bfs() : for队列
5. for队列 : 出队顶点。for遍历所有顶点 
6. for遍历所有顶点 ： 若定点未访问 且出队的顶点到该顶点有边，标记`visited[该顶点]`且该顶点入队
