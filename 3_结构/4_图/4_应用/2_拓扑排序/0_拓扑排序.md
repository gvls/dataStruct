##  拓扑排序
将有向图中的顶点生成一个线性序列，任意一个`<vx,vy> E E(G)` ，在序列中`vx` 出现在`vy` 之前
得到的结果不唯一

* 目标 
解决一个工程能否顺利完成
处理有向图得到 无环有向图 : 若原网顶点都被输出，原网不存在回路是AOV网，存在拓扑排序。若原网顶点输出少了，原网不是AOV网，不存在拓扑排序

* AOV网 
一个表示工程的**有向无环图**中，顶点表示活动， 弧表示活动之间的优先关系


* 使用 邻接表
```shell
[入度][data][边指针] ->
```


###   步骤
1. 查找 入度 == 0 的顶点
2. 删除 以该顶点为 弧尾的弧
3. 重复 1 ~ 2
