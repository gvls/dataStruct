##  顺序结构
* 栈顶指针
指向 最后一个元素 / 最后一个元素的后一个位置

* 栈底指针
指向 第一个元素 / 第一个元素的前一个位置

```shell
			add()   get()
				\   ^
				 v /
		 _		_   _
		/		|	|
	   /		|	| <- top
 size()		   /| x |
	  \	  len()	| x |
	   \	   \| x | <- bottom
		\-		-----
```
