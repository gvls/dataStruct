##  逻辑结构
数据元素之间的 关系

###   集合
所有元素有相同的特点
所有元素都是**平等的** 
```shell
 /--------------\
/	x		x	 \
|	   	x		 |
|	x  		x	 |
|x	   	x		 |
|	   			x|
\	 x 		 x	 /
 \--------------/
```

###   线性
关系：一对一
```shell
[X]-[X]-[X]-[X]-[X]-[X]-[X]
```

###   树
关系：一对多的 层次关系
```shell
		[X]
	   /   \
	 [X]    [X]
	/ \	   / \	\
 [X] [X]  [X][X][X]
		...
```

###   图形
关系：多对多
```shell
		[X]
	   / | \
	  /  |  \
	 /  [X]  \
	/  / |	\ \
    [X]-[X]-[X]
```
