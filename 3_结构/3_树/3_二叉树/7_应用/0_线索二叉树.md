##  线索二叉树
* 为了 
利用叶子结点的子树指针、减少遍历的时间
n 个结点的二叉树有 n+1 个空指针域

* 线索化 
二叉树 	== 某种遍历方式 ==>	 双向链表

* 线索 
叶子结点子树指针
左子树保存该结点的前驱，右子树保存该结点的后继

* 结点
添加字段
```shell
[左子树指针][左子树是否线索][data][右子树是否线索][右子树指针]
```


###   遍历双向链表
添加头结点
头结点的 左子树 链接 根结点
头结点的 右子树 链接 遍历的最后一个结点
遍历的第一个结点的 左子树 链接 头结点
遍历的最后一个结点的 右子树 指向 头结点
使用 前驱 和 后继 遍历 线索二叉树



###   适合的场景 
二叉树需要经常被遍历
查找结点时需要遍历序列中的 前驱和后继
